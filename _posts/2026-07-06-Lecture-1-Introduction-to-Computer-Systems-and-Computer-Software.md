## 2. What is Hardware?

Hardware refers to the physical parts of a computer that can be seen and touched.

Examples:

- Keyboard
- Mouse
- Monitor
- CPU Cabinet
- RAM
- Hard Disk
- SSD
- Printer
- Scanner
- Webcam

## 3. What is Software?

Software is a collection of programs, instructions, and related data that tells the computer what to do. Software cannot be touched physically. It exists in digital form.

## 4. Relationship Between Hardware and Software

![Relationship Between Hardware and Software]({{ '/assets/images/Relationship-Between-Hardware-and-Software.png' | relative_url }})

#### **Example:**

1. You click "Print."
2. Software sends instructions.
3. Printer (hardware) prints the document.

### # Characteristics of Software:

1. Intangible: Software cannot be touched.
2. Developed: Hardware is manufactured. Software is developed through programming.
3. Easy to Copy: A software program can be copied many times.
4. Requires Maintenance: Software needs updates and bug fixes.
5. Depends on Hardware: Software cannot run without hardware.
